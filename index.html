Formula 1 Race Circuit Information Service

Available API Endpoints

GET /api/circuits

Get all Formula 1 circuits. Returns an array of all circuit objects with complete information.

/api/circuits

Returns: Array of circuit objects

GET /api/circuits/:id

Get a specific circuit by ID. Returns a single circuit object with all details.

/api/circuits/1

Returns: Single circuit object
